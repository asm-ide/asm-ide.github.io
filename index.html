Features

App UI
  • material design
  • Project code
  • code API
  • real-time code highlighter
  • java code analysis plugin(local)
Project apkBuilder
  • compiler API
  • android APK builder
  • gradle support
Project block
  • making java/android app in block coding
  • block drag & drop
  • java code -> blocks, blocks -> java code converter (J2B, B2J)
